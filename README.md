Usage:

    $ NEWRELIC_APPNAME=example.org ./composer.phar install

To install:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/easybiblabs/composer-newrelic"
        }
    ],
    "require": {
        "easybib/composer-newrelic": "*"
    },
    "minimum-stability": "dev"
}
```
